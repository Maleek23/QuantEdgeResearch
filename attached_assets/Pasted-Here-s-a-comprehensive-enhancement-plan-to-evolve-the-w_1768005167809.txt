## **1. Core Architecture: From Static Scanner to Historical Database**

### **Current State**
- 45 symbols, 15-minute refresh, snapshot-only data
- No persistence of *your* additions/removals
- No historical tracking (what was NIO's grade in March?)

### **New Database Schema**
```sql
-- Multiple named watchlists per user/year
CREATE TABLE user_watchlists (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  name VARCHAR(255), -- "2026 Breakouts", "China Plays", "Nuclear Macro"
  year INT NOT NULL, -- Enables year-over-year analysis
  created_at TIMESTAMP DEFAULT NOW(),
  notes TEXT -- "Tracking for Biden policy impact"
);

-- Individual entries with lifecycle tracking
CREATE TABLE watchlist_entries (
  id UUID PRIMARY KEY,
  watchlist_id UUID REFERENCES user_watchlists(id),
  symbol VARCHAR(20),
  added_at TIMESTAMP,
  removed_at TIMESTAMP, -- NULL = currently watching
  added_grade VARCHAR(2), -- Grade when first added
  added_confidence INT,   -- Snapshot for performance tracking
  tags JSONB, -- ["breakout", "china", "volume_surge"]
  notes TEXT, -- "4-month high turnover Jan 2026"
  added_reason VARCHAR(50) -- "user_search", "scanner", "news"
);

-- Daily historical snapshots
CREATE TABLE watchlist_history (
  id UUID PRIMARY KEY,
  entry_id UUID REFERENCES watchlist_entries(id),
  date DATE,
  grade VARCHAR(2),
  confidence_score INT,
  technical_snapshot JSONB, -- Full RSI, ADX, volume data
  price DECIMAL(10,2),
  UNIQUE(entry_id, date)
);
```

## **2. Year-Long Tracking Features**

### **Historical Grade Timeline**
**UI Component**: Interactive timeline chart for each symbol

- **Y-axis**: Grade tier (S=100, A=85, B=70, etc.)
- **Events**: Clickable markers for earnings, news, your manual notes
- **Performance overlay**: Price change % since addition

**Implementation**:
```typescript
// server/watchlist-history-service.ts
async function getGradeHistory(symbol: string, year: 2026) {
  return db.query(`
    SELECT date, grade, confidence_score, price 
    FROM watchlist_history 
    WHERE symbol = $1 AND year = $2
    ORDER BY date ASC
  `);
}
```
