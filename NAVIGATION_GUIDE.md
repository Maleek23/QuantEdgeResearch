# QuantEdge Navigation Guide
**Last Updated:** 2026-01-25

## Overview
This document explains the unified navigation structure designed for clarity, trader confidence, and consistent user experience.

---

## Design Philosophy

### Quant Trader's Mental Model:
1. **"What should I trade today?"** ‚Üí Opportunities
2. **"What's happening?"** ‚Üí Markets
3. **"Tell me about THIS stock"** ‚Üí Stock Detail Page
4. **"I want to research"** ‚Üí Research
5. **"How am I doing?"** ‚Üí Portfolio

### UI/UX Principles:
- **Search-first**: Global search always visible, drives to Stock Detail
- **Context persistence**: Stock Context Bar shows after search
- **5 core pages max**: Like Yahoo Finance/Bloomberg
- **Clear hierarchy**: One purpose per page
- **Consistent design**: Aurora background, cyan accent, glass-morphism

---

## Navigation Structure

### Header Navigation (HeaderNav)
```
[Logo] | Opportunities | Markets | Research | Portfolio | [Search] | [User]
```

**Always Visible:**
- Logo (links to Opportunities if logged in, Landing if not)
- Main navigation (4 items)
- Global search bar
- User actions (notifications, logout)

**Active State:**
- Current page highlighted in cyan with background
- Icon + text on desktop, icon only on mobile

## The 5 Core Pages

### 1. **Opportunities** (`/trade-desk`)
**Value Prop:** "Your AI Research Team - 24/7"

**What it is:**
- AI bots proactively scanning markets
- Trade ideas generated by Quant Bot, AI Bot, Flow Bot, Penny Stock Bot
- Research Pulse showing bot activity
- Personal Edge Analytics

**Page Components:**
- Trade idea cards with bot badges
- Filter by: Bot type, Asset class, Time horizon, Conviction
- Sort by: Confidence, Recency, Performance
- "Why this trade?" explainers

**Navigation:**
- Icon: ‚ö° Zap
- Color: Purple/Electric blue
- Default page after login

**User Journey:**
1. Login ‚Üí See Opportunities
2. Click trade idea ‚Üí Go to Stock Detail
3. Review idea ‚Üí Take action

---

### 5. **Stock Detail** (NEW - not in main nav)
**URL:** `/stock/{SYMBOL}` or `/analyze?symbol={SYMBOL}`
**Value Prop:** "Everything About This Stock"

**What it is:**
- Unified page with ALL analysis types
- Reached via search or clicking opportunities
- Tabs for different analysis views

**Tabs:**
1. **Overview** - Price, stats, AI summary
2. **Chart** - Technical analysis (chart-analysis.tsx content)
3. **Options** - Options flow, Greeks (options-analyzer.tsx content)
4. **Smart Money** - Insider trades, institutions (smart-money.tsx content)
5. **AI Insights** - Sentiment, predictions (ai-stock-picker.tsx content)
6. **News** - Recent news and events

**Not in main navigation because:**
- Accessed dynamically (different stock each time)
- Stock Context Bar shows current stock
- Can return to previous page easily

## Stock Context Bar

**Appears:** After searching for a stock or clicking an opportunity

**Shows:**
- Stock symbol, name, price, change
- Quick action buttons:
  - Chart
  - Options
  - Smart Money
  - AI Analysis
- Close button

**Behavior:**
- Sticky below header
- Persists across page navigation
- Click action buttons ‚Üí Navigate to Stock Detail with appropriate tab

## User Flows

### Flow 1: New User Discovering Platform
1. Land on Landing page
2. Sign up
3. Redirected to **Opportunities** (see AI trade ideas)
4. Click idea ‚Üí **Stock Detail** (deep dive)
5. Want more context? ‚Üí **Markets** (check market conditions)

### Flow 2: Active Trader Morning Routine
1. Login ‚Üí **Opportunities** (see overnight ideas)
2. Check **Markets** (market open conditions)
3. Search specific stock ‚Üí **Stock Detail** (detailed analysis)
4. Check **Portfolio** (review positions)
5. Return to **Opportunities** (find new trades)

### Flow 3: Research-Focused User
1. Login ‚Üí Navigate to **Research**
2. Search stock
3. Select analysis agent
4. View results in **Stock Detail**
5. Save to watchlist
6. Repeat

### Flow 4: Performance Review
1. Navigate to **Portfolio**
2. Review P&L
3. Click underperforming position ‚Üí **Stock Detail**
4. Analyze "what went wrong"
5. Check **Opportunities** for better trades

## Design Tokens (Consistent Across Platform)

### Colors:
- **Primary:** Cyan (`rgb(6 182 212)`)
- **Background:** Slate 950 (`rgb(2 6 23)`)
- **Card:** Slate 900/90 with backdrop blur
- **Text Primary:** Slate 100
- **Text Secondary:** Slate 400
- **Success:** Green 400
- **Danger:** Red 400
- **Warning:** Amber 400

### Typography:
- **Headings:** Bold, tracking-tight
- **Body:** Regular, readable line height
- **Mono:** For code, data

### Effects:
- **Aurora Background:** On all pages
- **Glass-morphism:** `backdrop-blur-xl`, gradient borders
- **Hover:** Cyan glow, border highlight
- **Active:** Cyan background, brighter text
